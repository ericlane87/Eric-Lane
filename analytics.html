Portfolio Analytics

This page is intentionally not tracked. It is not linked from the public resume page and includes noindex so search engines should not list it.

Status

  • Public pages load analytics from analytics.js.
  • analytics.html is excluded, so visiting this page will not count as a portfolio view.
  • Real view counts start after the Cloudflare Web Analytics token is added in analytics-config.js.

Where To View Counts

  • Open your Cloudflare Web Analytics dashboard to see views by hour, last 24 hours, last 7 days, and last 30 days.
  • Use this page as your private shortcut/checklist: https://ericlane87.github.io/Eric-Lane/analytics.html.

Activation Needed

  • Create or open Cloudflare Web Analytics.
  • Add this site: ericlane87.github.io/Eric-Lane/.
  • Replace REPLACE_WITH_CLOUDFLARE_WEB_ANALYTICS_TOKEN in analytics-config.js with the token Cloudflare gives you.
